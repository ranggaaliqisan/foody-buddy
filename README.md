# FoodyBuddy

A landing page for FoodyBuddy - Indonesia's social food delivery platform that connects food lovers, restaurants, and F&B professionals.

## What It Does

- Showcases the platform concept
- Explains features (restaurant profiles, job board, food delivery, community groups)
- Collects emails for waitlist

## Tech Stack

- HTML
- CSS
- JavaScript
- Boxicons
- ScrollReveal.js
